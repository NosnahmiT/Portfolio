Consistency Updates for Mobile Navigation
UX/UI Professional Work

Over the past few years, my company has been working on a new design system. It was designed in tandem with a new system we have been developing for insurance agents. As our team has been maintaining and updating the design system we have been looking for opportunities where we can implement it in existing systems to improve consistency. We saw an opportunity to update the mobile navigation across multiple websites to make them consistent with the new design systema and with each other.

Our intranet, agent portal, and public website all had different navigation systems. There navigation is shown in the image above. The first two utilized our design system, but in different ways. The public website didn't match at all. Researching the best practices for mobile navigation, we decided to implement a sequential navigation system across all three sites. It seemed the best way to display several categories and subcategories on a small screen. The new menu is shown below implemented on our intranet.
